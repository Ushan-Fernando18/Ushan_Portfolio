Python Project

I developed a Python-based transaction management system with a user-friendly GUI, In order to facilitate effective data entry, editing, deletion, and retrieval, this project sought to optimize daily transaction operations. This project improved my knowledge of data structures, Python programming, and GUI design while giving me hands-on experience creating real-world applications.

Java Project

I developed a Java-based Student Management System to streamline the administration of student records. Effective administration of student enrollment, tracking, and performance evaluation is guaranteed by this system. It enables smooth data entry, editing, and retrieval through the use of strong data validation and an intuitive user interface. Additionally, the system has search features that allow users to find specific student data by name, ID, or module details. It also keeps track of student achievements all of which offer insightful information for academic analysis and decision-making. Through this project, I was able to improve my understanding of database administration, object-oriented programming, and Java programming while also gaining practical experience in developing a working application.
